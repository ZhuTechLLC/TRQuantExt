<div class="board-details">

        <li><strong>智能手机（2010-2015）：</strong>渗透率从20%到80%</li>

### 财务质量的深度分析

<div class="board-details">
  <div class="board-card">
    <h4>📈 ROE的分解（杜邦分析）</h4>
    <div class="board-content">
      <p><strong>ROE = 净利率 × 资产周转率 × 权益乘数</strong></p>
      <ul>
        <li><strong>高净利率型：</strong>茅台（净利率50%+），依靠品牌溢价</li>
        <li><strong>高周转型：</strong>永辉超市（周转率3+），依靠规模效应</li>
        <li><strong>高杠杆型：</strong>银行（权益乘数10+），需警惕风险</li>
      </ul>
      <p><strong>最佳组合：</strong>高净利率+适度周转+低杠杆</p>
    </div>
  </div>

  <div class="board-card">
    <h4>💰 现金流的重要性</h4>
    <div class="board-content">
      <p><strong>经营现金流 vs 净利润：</strong></p>
      <ul>
        <li><strong>健康状态：</strong>经营现金流 ≥ 净利润的80%</li>
        <li><strong>警惕信号：</strong>连续3年经营现金流 < 净利润的50%</li>
        <li><strong>危险信号：</strong>经营现金流持续为负</li>
      </ul>
      <p><strong>案例对比：</strong></p>
      <ul>
        <li>✅ 贵州茅台：经营现金流常年>净利润</li>
        <li>❌ 康得新：利润造假，现金流与利润严重背离</li>
      </ul>
    </div>
  </div>

### PEG估值法的应用

<div class="tip-box">
  <h4>💡 PEG = PE / 增长率</h4>
  <p><strong>解读：</strong></p>
  <ul>
    <li><strong>PEG < 1：</strong>相对低估，增长速度超过估值</li>
    <li><strong>PEG = 1：</strong>合理估值，增长与估值匹配</li>
    <li><strong>PEG > 2：</strong>相对高估，需要超预期增长</li>
  </ul>
  <p><strong>应用示例：</strong></p>
  <ul>
    <li>某公司PE 30倍，预期增速40% → PEG=0.75，相对低估</li>
    <li>某公司PE 50倍，预期增速20% → PEG=2.5，相对高估</li>
  </ul>
  <p><strong>注意事项：</strong>PEG适用于成长股，不适用于周期股和价值股</p>
</div>
