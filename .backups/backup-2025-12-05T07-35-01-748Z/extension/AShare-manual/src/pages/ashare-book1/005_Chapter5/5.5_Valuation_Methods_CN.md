<h2 id="section-5-5-1">📊 5.5.1 市盈率（PE）</h2>

### PE的定义与计算

<div class="tip-box">
  <h4>📊 PE = 股价 / 每股收益 = 市值 / 净利润</h4>
  <p><strong>含义：</strong>投资者愿意为1元利润支付多少钱</p>
  <p><strong>另一种理解：</strong>按当前利润，多少年能收回投资</p>
  <ul>
    <li><strong>PE = 20倍：</strong>20年收回投资（假设利润不变）</li>
    <li><strong>PE = 10倍：</strong>10年收回投资</li>
  </ul>
</div>

### PE的历史分位数

<div class="board-details">
  <div class="board-card">
    <h4>📊 如何使用PE分位数</h4>
    <div class="board-content">
      <p><strong>方法：</strong>计算当前PE在历史PE中的位置</p>
      <ul>
        <li><strong>0-20%分位：</strong>历史低估区间，可考虑买入</li>
        <li><strong>20-50%分位：</strong>合理偏低，可持有</li>
        <li><strong>50-80%分位：</strong>合理偏高，谨慎</li>
        <li><strong>80-100%分位：</strong>历史高估区间，考虑减仓</li>
      </ul>
    </div>
  </div>

  <div class="board-card">
    <h4>📈 案例：贵州茅台PE历史</h4>
    <div class="board-content">

<h2 id="section-5-5-2">📈 5.5.2 市净率（PB）</h2>

### PB的定义与应用

<div class="tip-box">
  <h4>📊 PB = 股价 / 每股净资产 = 市值 / 净资产</h4>
  <p><strong>含义：</strong>投资者愿意为1元净资产支付多少钱</p>
  <p><strong>适用场景：</strong></p>
  <ul>
    <li><strong>重资产行业：</strong>银行、地产、钢铁、电力</li>
    <li><strong>周期行业：</strong>利润波动大时PE失效</li>
    <li><strong>亏损公司：</strong>PE为负时用PB</li>
  </ul>
</div>

<h2 id="section-5-5-3">🎯 5.5.3 PEG估值法</h2>

### PEG的定义与计算

<div class="tip-box">
  <h4>📊 PEG = PE / 净利润增速（%）</h4>
  <p><strong>含义：</strong>估值与成长性的匹配程度</p>
  <p><strong>标准：</strong></p>
  <ul>
    <li><strong>PEG < 1：</strong>相对低估，增速超过估值</li>
    <li><strong>PEG = 1：</strong>合理估值，增速与估值匹配</li>
    <li><strong>PEG > 2：</strong>相对高估，估值透支增长</li>
  </ul>
  <p><strong>示例：</strong></p>
  <ul>
    <li>PE 30倍，增速40% → PEG = 0.75，低估</li>
    <li>PE 50倍，增速20% → PEG = 2.5，高估</li>
  </ul>
</div>

### PEG的局限性

<div class="board-details">
  <div class="board-card">
    <h4>⚠️ PEG的适用条件</h4>
    <div class="board-content">
      <ul>
        <li><strong>适用于成长股：</strong>增速>15%</li>
        <li><strong>不适用于周期股：</strong>利润波动大</li>
        <li><strong>不适用于价值股：</strong>增速<10%</li>
        <li><strong>增速需可持续：</strong>一次性增长不算</li>
      </ul>
    </div>
  </div>

  <div class="board-card">
    <h4>🔍 使用注意事项</h4>
    <div class="board-content">
      <ul>
        <li>增速应使用未来3年复合增速</li>
        <li>需要验证增速预测的可靠性</li>
        <li>结合行业特点调整标准</li>
        <li>高增速公司可接受较高PEG</li>
      </ul>
    </div>
  </div>
</div>

<h2 id="section-5-5-4">💵 5.5.4 DCF估值法</h2>

### DCF的基本原理

<div class="tip-box">
  <h4>📊 DCF = 未来现金流的现值之和</h4>
  <p><strong>核心思想：</strong>公司价值等于未来所有现金流折现到今天的价值</p>
  <p><strong>公式：</strong></p>
  <p>V = Σ FCF_t / (1+r)^t + TV / (1+r)^n</p>
  <ul>
    <li><strong>FCF_t：</strong>第t年的自由现金流</li>
    <li><strong>r：</strong>折现率（通常用WACC）</li>
    <li><strong>TV：</strong>终值（永续增长模型）</li>
  </ul>
</div>

### DCF的简化应用

<div class="board-details">
  <div class="board-card">
    <h4>📊 两阶段模型</h4>
    <div class="board-content">
      <p><strong>第一阶段：</strong>高速增长期（5-10年）</p>
      <ul>
        <li>预测每年的自由现金流</li>
        <li>增速可以较高（20-30%）</li>
      </ul>
      <p><strong>第二阶段：</strong>永续增长期</p>
      <ul>
        <li>假设永续增长率（2-3%）</li>
        <li>TV = FCF_n × (1+g) / (r-g)</li>
      </ul>
    </div>
  </div>

  <div class="board-card">
    <h4>🎯 关键假设</h4>
    <div class="board-content">
      <table>
        <tr>
          <td><strong>折现率</strong></td>
          <td>8-12%</td>
          <td>反映风险水平</td>
        </tr>
        <tr>
          <td><strong>永续增长率</strong></td>
          <td>2-3%</td>
          <td>不应超过GDP增速</td>
        </tr>
        <tr>
          <td><strong>预测期</strong></td>
          <td>5-10年</td>
          <td>太长不可靠</td>
        </tr>
      </table>
    </div>
  </div>
</div>

## 💎 核心要点总结

<div class="key-points">
  <div class="key-point">
    <h4>📊 PE是最常用的估值指标</h4>
    <p>需要结合行业特点和历史分位数使用。</p>
  </div>
  <div class="key-point">
    <h4>📈 PB适用于重资产和周期行业</h4>
    <p>PB与ROE结合使用更有意义。</p>
  </div>
  <div class="key-point">
    <h4>🎯 PEG是成长股的利器</h4>
    <p>PEG<1是相对低估的信号。</p>
  </div>
  <div class="key-point">
    <h4>💵 DCF是绝对估值的金标准</h4>
    <p>关键假设对结果影响很大，需谨慎使用。</p>
