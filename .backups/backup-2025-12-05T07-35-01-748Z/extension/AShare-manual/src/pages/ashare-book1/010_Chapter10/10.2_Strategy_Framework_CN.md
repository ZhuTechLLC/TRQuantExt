### 定性选股标准

<div class="board-details">
  <div class="board-card">
    <h4>🏢 商业模式分析</h4>
    <div class="board-content">
      <ul>
        <li><strong>简单易懂：</strong>商业模式清晰，能用一句话说明白</li>
        <li><strong>可预测性：</strong>收入和利润相对可预测</li>
        <li><strong>现金流好：</strong>能持续产生现金流</li>
        <li><strong>轻资产：</strong>优先选择轻资产模式</li>
      </ul>
    </div>
  </div>

  <div class="board-card">
    <h4>🛡️ 护城河分析</h4>
    <div class="board-content">
      <ul>
        <li><strong>品牌优势：</strong>如茅台、海天</li>
        <li><strong>成本优势：</strong>如牧原、海螺</li>
        <li><strong>网络效应：</strong>如腾讯、美团</li>
        <li><strong>转换成本：</strong>如用友、金蝶</li>
        <li><strong>规模效应：</strong>如顺丰、京东</li>
      </ul>
    </div>
  </div>

  <div class="board-card">
    <h4>👥 管理层评估</h4>
    <div class="board-content">
      <ul>
        <li><strong>诚信记录：</strong>历史上是否有诚信问题</li>
        <li><strong>持股比例：</strong>管理层是否有足够持股</li>
        <li><strong>分红记录：</strong>是否愿意分红回报股东</li>
        <li><strong>战略眼光：</strong>历史决策是否正确</li>
        <li><strong>执行能力：</strong>战略是否能有效执行</li>
      </ul>
    </div>
  </div>

  <div class="board-card">
    <h4>📈 行业地位</h4>
    <div class="board-content">
      <ul>
        <li><strong>市场份额：</strong>行业前三或细分龙头</li>
        <li><strong>行业趋势：</strong>行业处于成长期或成熟期</li>
        <li><strong>竞争格局：</strong>竞争格局稳定或优化</li>
        <li><strong>政策支持：</strong>符合国家政策方向</li>
      </ul>
    </div>
  </div>

### 负面清单（不买标准）

<div class="tip-box">
  <h4>❌ 负面清单——这些股票不买</h4>
  <ul>
    <li><strong>财务问题：</strong>连续亏损、财务造假嫌疑、审计保留意见</li>
    <li><strong>治理问题：</strong>大股东占款、频繁关联交易、管理层频繁变动</li>
    <li><strong>业务问题：</strong>主业不清、频繁跨界、概念炒作</li>
    <li><strong>估值问题：</strong>PE>100（非高成长）、PB>10（非轻资产）</li>
    <li><strong>流动性问题：</strong>日均成交额<1000万</li>
    <li><strong>政策风险：</strong>政策明确打压的行业</li>
  </ul>
</div>

### 卖出时机规则

<div class="tip-box">
  <h4>📤 卖出信号分类</h4>
  <p><strong>止盈信号：</strong></p>
  <ul>
    <li>估值处于历史高位（>80%分位）</li>
    <li>达到预设目标价</li>
    <li>PEG>2，成长性价比差</li>
    <li>发现更好的投资机会</li>
  </ul>
  <p><strong>止损信号：</strong></p>
  <ul>
    <li>亏损达到8-10%（固定止损）</li>
    <li>跌破关键支撑位（技术止损）</li>
    <li>投资逻辑被破坏（逻辑止损）</li>
    <li>持有6个月无起色（时间止损）</li>
  </ul>
  <p><strong>调仓信号：</strong></p>
  <ul>
    <li>行业集中度过高</li>
    <li>单只仓位过重</li>
    <li>有更好的机会需要资金</li>
  </ul>

### 建仓规则

<div class="tip-box">
  <h4>📈 分批建仓方法</h4>
  <p><strong>标准三批建仓法：</strong></p>
  <ul>
    <li><strong>第一批（30%）：</strong>初步确认机会，建立观察仓</li>
    <li><strong>第二批（40%）：</strong>逻辑验证，加仓到主要仓位</li>
    <li><strong>第三批（30%）：</strong>趋势确认，加仓到目标仓位</li>
  </ul>
  <p><strong>金字塔加仓法：</strong></p>
  <ul>
    <li>股价上涨确认趋势后加仓</li>
    <li>每次加仓量递减</li>
    <li>如：首次50%，第二次30%，第三次20%</li>
  </ul>
  <p><strong>倒金字塔加仓法：</strong></p>
  <ul>
    <li>股价下跌时分批买入</li>
    <li>每次加仓量递增</li>
    <li>适用于高确定性的价值股</li>
  </ul>
</div>

### 减仓规则

<div class="board-details">
  <div class="board-card">
    <h4>📉 分批减仓方法</h4>
    <div class="board-content">
      <p><strong>目标价减仓法：</strong></p>
      <ul>
        <li>达到目标价80%时，减仓30%</li>
        <li>达到目标价100%时，减仓40%</li>
        <li>达到目标价120%时，减仓30%或全部</li>
      </ul>
      <p><strong>移动止盈法：</strong></p>
      <ul>
        <li>盈利20%后，设置回撤10%止盈</li>
        <li>盈利50%后，设置回撤15%止盈</li>
        <li>盈利100%后，设置回撤20%止盈</li>
      </ul>
    </div>
  </div>

<h2 id="section-10-2-4">📋 10.2.4 策略整合示例</h2>

### 完整策略框架示例

<div class="tip-box">
  <h4>📋 价值成长策略框架示例</h4>
  <p><strong>一、选股规则：</strong></p>
  <ul>
    <li>ROE>15%且连续3年稳定</li>
    <li>净利润增速>20%</li>
    <li>资产负债率<60%</li>
    <li>经营现金流/净利润>80%</li>
    <li>PE分位<50%或PEG<1</li>
    <li>行业龙头或细分龙头</li>
    <li>商业模式简单易懂</li>
  </ul>
  <p><strong>二、买入规则：</strong></p>
  <ul>
    <li>满足选股标准</li>
    <li>估值处于合理或低估区间</li>
    <li>股价站上20日均线</li>
    <li>分3批建仓，首次30%</li>
  </ul>
  <p><strong>三、仓位规则：</strong></p>
  <ul>
    <li>总仓位60-80%</li>
    <li>单只仓位上限15%</li>
    <li>行业仓位上限30%</li>
    <li>持股6-10只</li>
  </ul>
  <p><strong>四、卖出规则：</strong></p>
  <ul>
    <li>估值>80%历史分位，分批减仓</li>
    <li>亏损8%止损</li>
    <li>投资逻辑破坏，立即卖出</li>
    <li>持有6个月无起色，考虑换股</li>
  </ul>
</div>

## 💎 核心要点总结

<div class="key-points">
  <div class="key-point">
    <h4>🔍 选股要有标准</h4>
    <p>定量+定性结合，建立清晰的选股标准和负面清单。</p>
  </div>
  <div class="key-point">
    <h4>⏰ 择时要有规则</h4>
    <p>买卖都有明确的触发条件，不凭感觉操作。</p>
  </div>
  <div class="key-point">
    <h4>📈 仓位要有限制</h4>
    <p>控制单只、行业、总仓位，分批建仓减仓。</p>
  </div>
  <div class="key-point">
    <h4>📋 策略要成体系</h4>
    <p>选股、择时、仓位规则整合成完整的策略框架。</p>
  </div>
</div>
