### 净值计算方法

<div class="tip-box">
  <h4>📊 如何计算账户净值</h4>
  <p><strong>简单净值法（适合无资金变动）：</strong></p>
  <ul>
    <li>净值 = 当日总市值 / 初始资金</li>
    <li>收益率 = (当日净值 - 1) × 100%</li>
  </ul>
  <p><strong>份额净值法（适合有资金变动）：</strong></p>
  <ul>
    <li>初始份额 = 初始资金（如100万份）</li>
    <li>入金时：新增份额 = 入金金额 / 当日净值</li>
    <li>出金时：减少份额 = 出金金额 / 当日净值</li>
    <li>当日净值 = 当日总市值 / 总份额</li>
  </ul>
  <p><strong>建议：</strong>使用Excel建立净值追踪表，或使用雪球组合功能</p>
</div>

<h2 id="section-10-5-2">📈 10.5.2 绩效评估指标</h2>

### 核心评估指标详解

<div class="comparison-table">
  <table>
    <thead>
      <tr>
        <th>指标</th>
        <th>计算公式</th>
        <th>优秀标准</th>
        <th>说明</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td><strong>绝对收益</strong></td>
        <td>期末净值/期初净值-1</td>
        <td>>0</td>
        <td>最基本的收益指标</td>
      </tr>
      <tr>
        <td><strong>年化收益</strong></td>
        <td>(1+收益率)^(365/天数)-1</td>
        <td>>15%</td>
        <td>统一时间维度对比</td>
      </tr>
      <tr>
        <td><strong>相对收益</strong></td>
        <td>组合收益-基准收益</td>
        <td>>0</td>
        <td>跑赢基准的能力</td>
      </tr>
      <tr>
        <td><strong>最大回撤</strong></td>
        <td>(峰值-谷值)/峰值</td>
        <td><25%</td>
        <td>风险控制能力</td>
      </tr>
      <tr>
        <td><strong>夏普比率</strong></td>
        <td>(年化收益-3%)/年化波动率</td>
        <td>>1</td>
        <td>风险调整后收益</td>
      </tr>
      <tr>
        <td><strong>卡玛比率</strong></td>
        <td>年化收益/最大回撤</td>
        <td>>1</td>
        <td>收益回撤比</td>
      </tr>
      <tr>
        <td><strong>胜率</strong></td>
        <td>盈利次数/总交易次数</td>
        <td>>50%</td>
        <td>选股成功率</td>
      </tr>
      <tr>
        <td><strong>盈亏比</strong></td>
        <td>平均盈利/平均亏损</td>
        <td>>2</td>
        <td>盈亏效率</td>
      </tr>
    </tbody>
  </table>
</div>

### 绩效评估维度

<div class="board-details">
  <div class="board-card">
    <h4>📈 多维度评估框架</h4>
    <div class="board-content">
      <p><strong>1. 收益维度：</strong></p>
      <ul>
        <li>绝对收益：是否赚钱</li>
        <li>相对收益：是否跑赢基准</li>
        <li>年化收益：长期收益水平</li>
      </ul>
      <p><strong>2. 风险维度：</strong></p>
      <ul>
        <li>最大回撤：最大亏损幅度</li>
        <li>波动率：收益的稳定性</li>
        <li>下行风险：只考虑亏损的风险</li>
      </ul>
      <p><strong>3. 效率维度：</strong></p>
      <ul>
        <li>夏普比率：单位风险收益</li>
        <li>卡玛比率：收益回撤比</li>
        <li>信息比率：主动风险收益</li>
      </ul>
      <p><strong>4. 交易维度：</strong></p>
      <ul>
        <li>胜率：选股成功率</li>
        <li>盈亏比：盈亏效率</li>
        <li>换手率：交易频率</li>
      </ul>
    </div>
  </div>
</div>

### 个股归因分析

<div class="tip-box">
  <h4>🔍 个股收益分析</h4>
  <p><strong>分析每只股票的贡献：</strong></p>
  <ul>
    <li><strong>盈利股票分析：</strong>
      <ul>
        <li>盈利原因是什么？（业绩、估值、市场）</li>
        <li>是否按计划买入和持有？</li>
        <li>有哪些可复制的经验？</li>
      </ul>
    </li>
    <li><strong>亏损股票分析：</strong>
      <ul>
        <li>亏损原因是什么？</li>
        <li>是选股错误还是时机错误？</li>
        <li>止损是否执行到位？</li>
        <li>有哪些需要改进的地方？</li>
      </ul>
    </li>
  </ul>
</div>

### 归因分析示例

<div class="board-details">
  <div class="board-card">
    <h4>📊 归因分析示例</h4>
    <div class="board-content">
      <p><strong>假设年度收益20%，分解如下：</strong></p>
      <ul>
        <li>基准收益（沪深300）：+8%</li>
        <li>超额收益：+12%</li>
      </ul>
      <p><strong>超额收益归因：</strong></p>
      <ul>
        <li>选股贡献：+15%（选对了茅台、宁德）</li>
        <li>择时贡献：-2%（几次调仓时机不佳）</li>
        <li>交易成本：-1%（佣金、印花税）</li>
        <li>合计超额：+12%</li>
      </ul>
      <p><strong>结论：</strong>选股能力强，择时能力需改进</p>
    </div>
  </div>
</div>

<div class="tip-box">

    <li>期初净值：______</li>

    <li>沪深300收益：______%</li>
    <li>超额收益：______%</li>
  </ul>
  <p><strong>二、持仓情况</strong></p>
  <ul>
    <li>持股数量：______只</li>
    <li>总仓位：______%</li>

  </ul>
  <p><strong>三、交易情况</strong></p>
  <ul>
    <li>本月买入：______笔</li>
    <li>本月卖出：______笔</li>

    <li>盈亏比：______</li>
  </ul>
  <p><strong>四、归因分析</strong></p>
  <ul>
    <li>贡献最大的股票：______</li>
    <li>亏损最大的股票：______</li>

  </ul>
  <p><strong>五、下月计划</strong></p>
  <ul>
    <li>关注机会：______</li>
