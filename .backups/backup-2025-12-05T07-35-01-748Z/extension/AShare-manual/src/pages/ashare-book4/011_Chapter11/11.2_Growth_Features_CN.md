### 利润增速与营收增速关系

<div class="board-details">
  <div class="board-card">
    <h4>📊 利润弹性分析</h4>
    <div class="board-content">
      <p><strong>利润增速/营收增速 > 1.5：</strong>经营杠杆效应明显，高倍股中占比60%</p>
      <p><strong>利润增速/营收增速 = 1-1.5：</strong>稳健增长，高倍股中占比30%</p>
      <p><strong>利润增速/营收增速 < 1：</strong>增收不增利，高倍股中占比仅10%</p>
      <p><strong>关键指标：</strong>利润增速持续高于营收增速是高倍股的重要特征</p>

<div class="tip-box">
  <h4>📈 ROE特征总结</h4>
  <ul>
    <li><strong>起点要求：</strong>启动时ROE > 12%，最好 > 15%</li>
    <li><strong>提升趋势：</strong>ROE同比提升是重要信号</li>
    <li><strong>杜邦分解：</strong>关注ROE提升的驱动因素（利润率/周转率/杠杆）</li>
    <li><strong>可持续性：</strong>利润率驱动的ROE提升更可持续</li>

<h2 id="screening">🎯 成长因子筛选标准</h2>

### 韬睿量化平台成长因子筛选

<div class="board-details">
  <div class="board-card">
    <h4>📊 成长因子筛选代码</h4>
    <div class="board-content">

```python
# 韬睿量化平台成长因子筛选
def growth_factor_screening(date):
    """
    成长因子筛选高倍股候选
    """
    # 获取所有股票
    stocks = get_all_securities()
    
    # 获取财务数据
    financials = get_fundamentals(query(
        valuation.code,
        indicator.inc_revenue_year_on_year,  # 营收增速
        indicator.inc_net_profit_year_on_year,  # 净利润增速
        indicator.roe,  # ROE
        indicator.inc_return  # ROE同比变化
    ).filter(
        valuation.code.in_(stocks)
    ), date=date)
    
    # 筛选条件
    candidates = financials[
        (financials['inc_revenue_year_on_year'] > 20) &  # 营收增速 > 20%
        (financials['inc_net_profit_year_on_year'] > 25) &  # 净利润增速 > 25%
        (financials['roe'] > 15) &  # ROE > 15%
        (financials['inc_return'] > 0)  # ROE同比提升
    ]
    
    return candidates['code'].tolist()
```
