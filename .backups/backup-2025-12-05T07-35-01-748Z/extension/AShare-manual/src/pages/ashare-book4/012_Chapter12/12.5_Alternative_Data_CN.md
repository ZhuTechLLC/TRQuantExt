```python
# 韬睿量化平台行业景气因子
def industry_prosperity_factor(industry, date):
    """
    行业景气度因子
    """
    # PMI数据
    pmi = get_macro_data('PMI', date)
    
    # 行业价格指数
    price_index = get_industry_price(industry, date)
    
    # 行业销量数据
    sales = get_industry_sales(industry, date)
    
    # 计算景气度得分
    prosperity_score = (
        pmi['value'] * 0.3 +
        price_index['change'] * 0.4 +
        sales['growth'] * 0.3
    )
    
    return prosperity_score
```

## 📈 另类数据应用建议

<div class="tip-box">
  <h4>💡 另类数据使用建议</h4>
  <ul>
    <li><strong>数据质量：</strong>确保数据来源可靠、更新及时</li>
    <li><strong>信号验证：</strong>另类数据信号需要回测验证</li>
    <li><strong>结合传统：</strong>另类数据应与传统因子结合使用</li>
    <li><strong>成本考量：</strong>部分另类数据获取成本较高</li>
    <li><strong>合规性：</strong>确保数据获取和使用合规</li>
  </ul>
