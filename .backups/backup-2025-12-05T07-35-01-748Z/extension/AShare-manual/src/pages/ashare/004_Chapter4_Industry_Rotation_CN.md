# 第四章：行业轮动与板块分析

> 行业选择是投资成功的重要因素。本章介绍A股行业轮动的规律、景气度分析方法和行业配置策略。

## 4.1 行业轮动的重要性

### 4.1.1 为什么行业选择重要

研究表明，投资收益的来源可以分解为：

| 收益来源 | 贡献度 | 说明 |
|----------|--------|------|
| 市场Beta | 40-50% | 大盘涨跌 |
| 行业选择 | 30-40% | 行业相对表现 |
| 个股选择 | 15-25% | 个股Alpha |

**结论**：选对行业比选对个股更重要

### 4.1.2 A股行业轮动特点

- **政策驱动明显**：产业政策对行业影响巨大
- **轮动速度快**：热点切换频繁
- **风格分化大**：成长/价值、大盘/小盘分化显著
- **资金效应强**：增量资金决定主线方向

## 4.2 经济周期与行业轮动

### 4.2.1 美林时钟框架

```
┌─────────────────────────────────────────────────────┐
│                  经济周期与行业轮动                    │
├─────────────────────────────────────────────────────┤
│                                                     │
│    复苏期                        过热期              │
│    ├─ 经济↑ 通胀↓               ├─ 经济↑ 通胀↑      │
│    ├─ 利率低位                   ├─ 利率上行         │
│    └─ 配置：周期、金融           └─ 配置：资源、能源  │
│                                                     │
│    衰退期                        滞胀期              │
│    ├─ 经济↓ 通胀↓               ├─ 经济↓ 通胀↑      │
│    ├─ 利率下行                   ├─ 利率高位         │
│    └─ 配置：防御、高股息         └─ 配置：现金、黄金  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 4.2.2 A股行业轮动路径

典型的A股行业轮动路径：

```
牛市初期 → 券商、银行、地产
    ↓
牛市中期 → 科技、新能源、军工
    ↓
牛市末期 → 消费、医药、周期
    ↓
熊市初期 → 公用事业、高股息
    ↓
熊市末期 → 超跌反弹、题材股
```

## 4.3 行业景气度分析

### 4.3.1 景气度指标体系

| 类别 | 指标 | 数据来源 |
|------|------|----------|
| 盈利能力 | 行业ROE、毛利率 | 财报数据 |
| 成长性 | 营收/利润增速 | 财报数据 |
| 估值水平 | PE/PB分位数 | 行情数据 |
| 资金流向 | 北向资金、主力资金 | 交易数据 |
| 分析师预期 | 盈利预测调整 | 研报数据 |

### 4.3.2 景气度评分模型

```python
# 景气度评分示例
景气度得分 = (
    0.25 × 营收增速分位数 +
    0.25 × 利润增速分位数 +
    0.20 × ROE变化分位数 +
    0.15 × 估值分位数(反向) +
    0.15 × 资金流向分位数
)
```

### 4.3.3 景气度拐点识别

关注以下信号识别景气度拐点：

**向上拐点信号**：
- 龙头公司订单/营收超预期
- 产品价格触底回升
- 库存周期见底
- 政策利好出台

**向下拐点信号**：
- 龙头公司业绩不及预期
- 产品价格下跌
- 产能过剩加剧
- 政策收紧

## 4.4 主要行业分析框架

### 4.4.1 消费行业

**核心关注点**：
- 消费升级趋势
- 品牌力和渠道力
- 提价能力
- 人口结构变化

**代表行业**：白酒、食品饮料、家电、医美

### 4.4.2 科技行业

**核心关注点**：
- 技术创新周期
- 国产替代进程
- 下游需求景气度
- 研发投入强度

**代表行业**：半导体、消费电子、软件、AI

### 4.4.3 周期行业

**核心关注点**：
- 产能周期位置
- 供需缺口
- 大宗商品价格
- 宏观经济周期

**代表行业**：有色、化工、钢铁、煤炭

### 4.4.4 金融行业

**核心关注点**：
- 利率环境
- 信贷周期
- 资产质量
- 监管政策

**代表行业**：银行、保险、券商

## 4.5 行业配置策略

### 4.5.1 核心-卫星策略

```
┌─────────────────────────────────────────────────────┐
│                  行业配置框架                         │
├─────────────────────────────────────────────────────┤
│                                                     │
│  核心仓位（60-70%）                                  │
│  ├─ 长期看好的行业                                  │
│  ├─ 龙头公司为主                                    │
│  └─ 低换手、长期持有                                │
│                                                     │
│  卫星仓位（30-40%）                                  │
│  ├─ 景气度上行的行业                                │
│  ├─ 短期催化的板块                                  │
│  └─ 灵活调整、捕捉轮动                              │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 4.5.2 行业轮动策略

```
每月/每季度执行:
1. 计算各行业景气度得分
2. 排序选取前N个行业
3. 在入选行业中选择个股
4. 调整持仓权重
```

### 4.5.3 行业中性策略

- 在每个行业内选择相对强势的个股
- 行业权重与基准保持一致
- 追求个股Alpha而非行业Beta
