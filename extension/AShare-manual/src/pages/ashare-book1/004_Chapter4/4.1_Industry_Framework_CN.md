<h2 id="section-4-1-1">📊 4.1.1 波特五力模型</h2>

### 五力模型概述

波特五力模型是分析行业竞争格局的经典框架，由哈佛商学院教授迈克尔·波特提出。五种力量共同决定了行业的盈利能力。

<div class="board-details">
  <div class="board-card">
    <h4>🏢 现有竞争者的竞争程度</h4>
    <div class="board-content">
      <p><strong>关键问题：</strong></p>
      <ul>
        <li>行业集中度如何？CR5是多少？</li>
        <li>竞争是价格战还是差异化竞争？</li>
        <li>行业增速如何？是存量博弈还是增量市场？</li>
        <li>退出壁垒高吗？</li>
      </ul>
      <p><strong>A股案例：</strong></p>
      <ul>
        <li><strong>高竞争：</strong>光伏组件（价格战激烈）</li>
        <li><strong>低竞争：</strong>高端白酒（品牌差异化明显）</li>
      </ul>
    </div>
  </div>

  <div class="board-card">
    <h4>🚪 新进入者的威胁</h4>
    <div class="board-content">
      <p><strong>关键问题：</strong></p>
      <ul>
        <li>进入壁垒有多高？（资金、技术、牌照）</li>
        <li>规模经济是否明显？</li>
        <li>品牌忠诚度如何？</li>
        <li>是否有专利保护？</li>
      </ul>
      <p><strong>A股案例：</strong></p>
      <ul>
        <li><strong>高壁垒：</strong>银行（牌照）、半导体设备（技术）</li>
        <li><strong>低壁垒：</strong>餐饮、零售</li>
      </ul>
    </div>
  </div>

  <div class="board-card">
    <h4>🔄 替代品的威胁</h4>
    <div class="board-content">
      <p><strong>关键问题：</strong></p>
      <ul>
        <li>是否存在替代产品？</li>
        <li>替代品的性价比如何？</li>
        <li>客户转换成本高吗？</li>
      </ul>
      <p><strong>A股案例：</strong></p>
      <ul>
        <li><strong>高替代威胁：</strong>燃油车（被新能源车替代）</li>
        <li><strong>低替代威胁：</strong>茅台（无替代品）</li>
      </ul>
    </div>
  </div>

  <div class="board-card">
    <h4>📦 供应商的议价能力</h4>
    <div class="board-content">
      <p><strong>关键问题：</strong></p>
      <ul>
        <li>供应商集中度如何？</li>
        <li>是否有替代供应商？</li>
        <li>转换供应商的成本高吗？</li>
      </ul>
      <p><strong>A股案例：</strong></p>
      <ul>
        <li><strong>供应商强势：</strong>芯片设计公司（依赖台积电）</li>
        <li><strong>供应商弱势：</strong>超市（供应商分散）</li>
      </ul>
    </div>
  </div>

  <div class="board-card">
    <h4>🛒 客户的议价能力</h4>
    <div class="board-content">
      <p><strong>关键问题：</strong></p>
      <ul>
        <li>客户集中度如何？</li>
        <li>客户转换成本高吗？</li>
        <li>产品差异化程度如何？</li>
      </ul>
      <p><strong>A股案例：</strong></p>
      <ul>
        <li><strong>客户强势：</strong>苹果供应链（依赖苹果）</li>

### 渗透率曲线：识别最佳投资窗口

<div class="tip-box">
  <h4>💡 渗透率与投资机会</h4>
  <p><strong>S曲线规律：</strong>新技术/新产品的渗透率通常呈S曲线形态</p>
  <ul>
    <li><strong>0-10%：</strong>导入期，技术验证阶段，高风险</li>
    <li><strong>10-30%：</strong>加速渗透期，最佳投资窗口，确定性提升</li>
    <li><strong>30-50%：</strong>快速普及期，仍有机会，但估值可能偏高</li>
    <li><strong>50-80%：</strong>成熟期，增速放缓，估值回归</li>
    <li><strong>>80%：</strong>饱和期，增长空间有限</li>
  </ul>
  <p><strong>案例：新能源车渗透率</strong></p>
  <ul>
    <li>2019年：5%（导入期末端）</li>
    <li>2021年：15%（加速渗透期）→ 宁德时代、比亚迪暴涨</li>
    <li>2024年：35%（快速普及期）→ 估值回归</li>
  </ul>
</div>

<h2 id="section-4-1-3">🔗 4.1.3 产业链分析</h2>

### 产业链价值分配

<div class="board-details">
  <div class="board-card">
    <h4>⬆️ 上游：资源与材料</h4>
    <div class="board-content">
      <p><strong>特点：</strong></p>
      <ul>
        <li>资源属性强，周期性明显</li>
        <li>毛利率受大宗商品价格影响</li>
        <li>进入壁垒：资源获取</li>
      </ul>
      <p><strong>案例：</strong>锂矿（天齐锂业）、铜矿（紫金矿业）</p>
      <p><strong>投资要点：</strong>关注资源价格周期</p>
    </div>
  </div>

  <div class="board-card">
    <h4>⏺️ 中游：制造与加工</h4>
    <div class="board-content">
      <p><strong>特点：</strong></p>
      <ul>
        <li>规模效应明显</li>
        <li>两头受挤压（上游涨价、下游压价）</li>
        <li>进入壁垒：技术和规模</li>
      </ul>
      <p><strong>案例：</strong>动力电池（宁德时代）、光伏电池（隆基绿能）</p>
      <p><strong>投资要点：</strong>关注成本控制和技术迭代</p>
    </div>
  </div>

  <div class="board-card">
    <h4>⬇️ 下游：品牌与渠道</h4>
    <div class="board-content">
      <p><strong>特点：</strong></p>
      <ul>
        <li>品牌溢价高</li>
        <li>直接面对消费者</li>
        <li>进入壁垒：品牌和渠道</li>
      </ul>
      <p><strong>案例：</strong>新能源车（比亚迪）、白酒（茅台）</p>
      <p><strong>投资要点：</strong>关注品牌力和市场份额</p>
    </div>
  </div>
</div>

<h2 id="section-4-1-4">🎯 4.1.4 行业研究清单</h2>

### 行业研究模板

<div class="tip-box">
  <h4>📋 行业研究必答问题</h4>
  <p><strong>一、行业基本面</strong></p>
  <ol>
    <li>行业规模多大？年增速多少？</li>
    <li>行业处于生命周期的哪个阶段？</li>
    <li>渗透率是多少？天花板在哪里？</li>
    <li>行业的核心驱动因素是什么？</li>
  </ol>
  <p><strong>二、竞争格局</strong></p>
  <ol>
    <li>CR5是多少？格局是分散还是集中？</li>
    <li>龙头公司是谁？市占率变化趋势？</li>
    <li>竞争的核心要素是什么？（成本、技术、品牌）</li>
    <li>新进入者威胁大吗？</li>
  </ol>
  <p><strong>三、产业链</strong></p>
  <ol>
    <li>产业链各环节的利润分配如何？</li>
    <li>哪个环节议价能力最强？</li>
    <li>上下游的依赖关系如何？</li>
  </ol>
  <p><strong>四、政策与风险</strong></p>
  <ol>
    <li>有哪些支持政策？</li>
    <li>有哪些监管风险？</li>
    <li>技术迭代风险大吗？</li>
  </ol>
</div>
