<h2 id="section-6-3-1">🏢 6.3.1 商业模式评估</h2>

### 商业模式评估框架

  <table>
    <thead>
      <tr>
        <th>评估维度</th>
        <th>核心问题</th>

    </thead>
    <tbody>
      <tr>
        <td><strong>价值创造</strong></td>
        <td>为客户解决什么问题？</td>
        <td>刚需、高频、不可替代</td>
        <td>茅台（社交需求）</td>
      </tr>
      <tr>
        <td><strong>收入模式</strong></td>
        <td>如何从客户获得收入？</td>
        <td>持续性、可预测性强</td>

      </tr>
      <tr>
        <td><strong>成本结构</strong></td>

  </table>

### 商业模式类型对比

<div class="board-details">
  <div class="board-card">
    <h4>✅ 优质商业模式</h4>
    <div class="board-content">
      <p><strong>1. 品牌消费品</strong></p>
      <ul>
        <li>特点：品牌溢价、复购率高</li>
        <li>案例：茅台、海天、伊利</li>
        <li>优势：提价能力强、现金流好</li>
      </ul>
      <p><strong>2. 平台/网络效应</strong></p>
      <ul>
        <li>特点：用户越多价值越高</li>
        <li>案例：腾讯、美团（港股）</li>
        <li>优势：赢家通吃、边际成本低</li>
      </ul>
      <p><strong>3. 连锁复制</strong></p>
      <ul>
        <li>特点：标准化、可快速扩张</li>
        <li>案例：爱尔眼科、通策医疗</li>
        <li>优势：增长确定性高</li>
      </ul>
    </div>
  </div>

  <div class="board-card">
    <h4>⚠️ 一般商业模式</h4>
    <div class="board-content">
      <p><strong>1. 制造业</strong></p>
      <ul>
        <li>特点：重资产、周期性</li>
        <li>案例：一般制造企业</li>
        <li>风险：产能过剩、价格战</li>
      </ul>
      <p><strong>2. 工程项目</strong></p>
      <ul>
        <li>特点：垫资、应收账款多</li>
        <li>案例：建筑公司</li>
        <li>风险：现金流差、坏账风险</li>
      </ul>
      <p><strong>3. 贸易/分销</strong></p>
      <ul>
        <li>特点：低毛利、资金占用大</li>
        <li>案例：贸易公司</li>
        <li>风险：议价能力弱</li>
      </ul>
    </div>
  </div>
</div>

### 商业模式评分标准

<div class="tip-box">
  <h4>📊 商业模式评分卡（满分5分）</h4>
  <table>
    <tr>
      <td><strong>可复制性</strong></td>
      <td>0-1分</td>
      <td>模式能否快速复制扩张</td>
    </tr>
    <tr>
      <td><strong>边际成本</strong></td>
      <td>0-1分</td>
      <td>规模扩张成本是否递减</td>
    </tr>
    <tr>
      <td><strong>客户粘性</strong></td>
      <td>0-1分</td>
      <td>客户转换成本高低</td>
    </tr>
    <tr>
      <td><strong>现金流质量</strong></td>
      <td>0-1分</td>
      <td>是否预收款模式</td>
    </tr>
    <tr>
      <td><strong>资产轻重</strong></td>
      <td>0-1分</td>
      <td>是否轻资产运营</td>
    </tr>
  </table>
  <p><strong>筛选标准：</strong>商业模式评分 ≥ 3分</p>
</div>

<h2 id="section-6-3-2">🏆 6.3.2 竞争优势评估</h2>

### 护城河类型识别

<div class="exchange-grid">
  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">🏷️</span>
      <h3>品牌护城河</h3>
    </div>
    <div class="exchange-content">
      <p><strong>识别特征：</strong></p>
      <ul>
        <li>可以比竞品卖更高价格</li>
        <li>客户主动寻找购买</li>
        <li>品牌知名度高</li>
      </ul>
      <p><strong>验证方法：</strong></p>
      <ul>
        <li>毛利率是否高于同行</li>
        <li>是否有提价能力</li>
        <li>市场份额是否稳定</li>
      </ul>
      <p><strong>A股案例：</strong>茅台、五粮液、海天、伊利</p>
    </div>
  </div>

  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">🔄</span>
      <h3>转换成本</h3>
    </div>
    <div class="exchange-content">
      <p><strong>识别特征：</strong></p>
      <ul>
        <li>客户更换供应商成本高</li>
        <li>客户流失率低</li>
        <li>合同期限长</li>
      </ul>
      <p><strong>验证方法：</strong></p>
      <ul>
        <li>客户续约率</li>
        <li>替换的时间和金钱成本</li>
        <li>客户集中度</li>
      </ul>
      <p><strong>A股案例：</strong>用友、恒生电子、广联达</p>
    </div>
  </div>

  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">💰</span>
      <h3>成本优势</h3>
    </div>
    <div class="exchange-content">
      <p><strong>识别特征：</strong></p>
      <ul>
        <li>成本显著低于竞争对手</li>
        <li>规模效应明显</li>
        <li>技术/工艺领先</li>
      </ul>
      <p><strong>验证方法：</strong></p>
      <ul>
        <li>毛利率对比同行</li>
        <li>市场份额趋势</li>
        <li>产能利用率</li>
      </ul>
      <p><strong>A股案例：</strong>万华化学、隆基绿能、宁德时代</p>
    </div>
  </div>

  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">🔗</span>
      <h3>网络效应</h3>
    </div>
    <div class="exchange-content">
      <p><strong>识别特征：</strong></p>
      <ul>
        <li>用户越多，产品价值越高</li>
        <li>市场份额高度集中</li>
        <li>用户增长自我强化</li>
      </ul>
      <p><strong>验证方法：</strong></p>
      <ul>
        <li>用户数量和活跃度</li>
        <li>市场份额</li>
        <li>用户获取成本趋势</li>
      </ul>
      <p><strong>A股案例：</strong>腾讯（港股）、分众传媒</p>
    </div>
  </div>
</div>

  <table>

      <tr>

      <tr>

      </tr>
    </tbody>
  </table>
</div>

<h2 id="section-6-3-3">👔 6.3.3 管理层评估</h2>

### 管理层评估四维度

<div class="board-details">
  <div class="board-card">
    <h4>🎯 战略能力</h4>
    <div class="board-content">
      <p><strong>评估要点：</strong></p>
      <ul>
        <li>对行业趋势的判断是否准确</li>
        <li>战略选择是否正确</li>
        <li>资源配置是否合理</li>
        <li>是否有前瞻性布局</li>
      </ul>
      <p><strong>验证方法：</strong></p>
      <ul>
        <li>回顾历史战略决策的效果</li>
        <li>对比同行的战略选择</li>
      </ul>
      <p><strong>正面案例：</strong>比亚迪王传福（早期押注新能源）</p>
      <p><strong>反面案例：</strong>乐视贾跃亭（过度扩张）</p>
    </div>
  </div>

  <div class="board-card">
    <h4>✅ 执行力</h4>
    <div class="board-content">
      <p><strong>评估要点：</strong></p>
      <ul>
        <li>制定的目标是否能达成</li>
        <li>业绩指引的准确性</li>
        <li>项目推进的效率</li>
      </ul>
      <p><strong>验证方法：</strong></p>
      <ul>
        <li>对比历史业绩指引与实际完成</li>
        <li>项目进度是否按计划推进</li>
      </ul>
      <p><strong>正面案例：</strong>宁德时代（产能扩张执行力强）</p>
    </div>
  </div>

  <div class="board-card">
    <h4>🤝 诚信正直</h4>
    <div class="board-content">
      <p><strong>评估要点：</strong></p>
      <ul>
        <li>信息披露是否透明</li>
        <li>是否有财务造假历史</li>
        <li>对小股东是否负责</li>
      </ul>
      <p><strong>警示信号：</strong></p>
      <ul>
        <li>频繁关联交易</li>
        <li>高管大规模减持</li>
        <li>频繁更换审计师</li>
        <li>业绩预告频繁修正</li>
      </ul>
    </div>
  </div>

  <div class="board-card">
    <h4>💰 股东回报意识</h4>
    <div class="board-content">
      <p><strong>评估要点：</strong></p>
      <ul>
        <li>分红政策是否稳定</li>
        <li>是否有回购计划</li>
        <li>再融资是否合理</li>
      </ul>
      <p><strong>验证方法：</strong></p>
      <ul>
        <li>历史分红率</li>
        <li>回购金额和频率</li>
        <li>增发次数和用途</li>
      </ul>
      <p><strong>正面案例：</strong>格力电器（高分红+回购）</p>
    </div>
  </div>
</div>

### 定性筛选检查清单

<div class="tip-box">
  <h4>✅ 定性筛选检查清单</h4>
  <p><strong>商业模式（必须满足）：</strong></p>
  <ul>
    <li>☐ 商业模式清晰，能用一句话说明白</li>
    <li>☐ 有明确的盈利模式和增长逻辑</li>
    <li>☐ 现金流特征良好</li>
  </ul>
  <p><strong>竞争优势（至少满足1项）：</strong></p>
  <ul>
    <li>☐ 品牌溢价能力强</li>
    <li>☐ 客户转换成本高</li>
    <li>☐ 成本优势明显</li>
    <li>☐ 网络效应显著</li>
  </ul>
  <p><strong>管理层（不能有红线）：</strong></p>
  <ul>
    <li>☐ 无财务造假历史</li>
    <li>☐ 无大规模关联交易</li>
    <li>☐ 高管未大规模减持</li>
    <li>☐ 分红政策稳定</li>
  </ul>
  <p><strong>公司治理（不能有红线）：</strong></p>
  <ul>
    <li>☐ 股权结构合理</li>
    <li>☐ 无信息披露违规</li>
    <li>☐ 大股东质押比例<50%</li>
  </ul>
</div>

## 💎 核心要点总结

<div class="key-points">
  <div class="key-point">
    <h4>🏢 商业模式决定天花板</h4>
    <p>可复制、边际成本低、现金流好是优秀商业模式的标志。</p>
  </div>
  <div class="key-point">
    <h4>🏆 护城河决定持久性</h4>
    <p>品牌、转换成本、成本优势、网络效应是四大护城河类型。</p>
  </div>
  <div class="key-point">
    <h4>👔 管理层是关键变量</h4>
    <p>战略能力、执行力、诚信正直是评估管理层的核心维度。</p>
  </div>
  <div class="key-point">
    <h4>⚖️ 治理是安全底线</h4>
    <p>股权结构合理、关联交易少、信息披露透明是基本要求。</p>
  </div>
</div>
