<h2 id="section-9-5-1">🔄 9.5.1 PDCA循环</h2>

### 投资中的PDCA应用

<div class="tip-box">
  <h4>🔄 PDCA循环详解</h4>
  <p><strong>P - Plan（计划）：</strong></p>
  <ul>
    <li>制定投资策略和交易计划</li>
    <li>明确选股标准和买卖规则</li>
    <li>设定收益目标和风险限制</li>
  </ul>
  <p><strong>D - Do（执行）：</strong></p>
  <ul>
    <li>按计划严格执行投资操作</li>
    <li>记录每笔交易的详细信息</li>
    <li>保持交易日志的完整性</li>
  </ul>
  <p><strong>C - Check（检查）：</strong></p>
  <ul>
    <li>定期复盘评估执行效果</li>
    <li>对比计划与实际的差异</li>
    <li>分析成功和失败的原因</li>
  </ul>
  <p><strong>A - Act（改进）：</strong></p>
  <ul>
    <li>总结经验教训</li>
    <li>优化策略和规则</li>
    <li>进入下一个循环</li>
  </ul>
</div>

### PDCA实践详解

<div class="comparison-table">
  <table>
    <thead>
      <tr>
        <th>阶段</th>
        <th>关键动作</th>
        <th>输出成果</th>
        <th>时间建议</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td><strong>Plan</strong></td>
        <td>研究分析、制定计划、设定目标</td>
        <td>投资策略文档、交易计划表</td>
        <td>周末或月初</td>
      </tr>
      <tr>
        <td><strong>Do</strong></td>
        <td>严格执行、记录过程、控制情绪</td>
        <td>交易记录、操作日志、情绪日记</td>
        <td>交易日实时</td>
      </tr>
      <tr>
        <td><strong>Check</strong></td>
        <td>对比分析、发现问题、归因分析</td>
        <td>复盘报告、问题清单、改进建议</td>
        <td>周末或月末</td>
      </tr>
      <tr>
        <td><strong>Act</strong></td>
        <td>总结经验、优化策略、更新规则</td>
        <td>改进措施、新版策略、知识库更新</td>
        <td>复盘后立即</td>
      </tr>
    </tbody>
  </table>
</div>

### PDCA循环示例

<div class="board-details">
  <div class="board-card">
    <h4>📊 月度PDCA示例</h4>
    <div class="board-content">
      <p><strong>Plan（月初）：</strong></p>
      <ul>
        <li>本月目标：收益5%，最大回撤<10%</li>
        <li>关注行业：消费、医药</li>
        <li>仓位计划：60-80%</li>
      </ul>
      <p><strong>Do（月中）：</strong></p>
      <ul>
        <li>买入贵州茅台、恒瑞医药</li>
        <li>记录买入理由、价格、仓位</li>
        <li>执行止损规则</li>
      </ul>
      <p><strong>Check（月末）：</strong></p>
      <ul>
        <li>实际收益3%，未达目标</li>
        <li>茅台盈利8%，恒瑞亏损5%</li>
        <li>分析：恒瑞买入时机不佳</li>
      </ul>
      <p><strong>Act（下月初）：</strong></p>
      <ul>
        <li>改进：增加技术面确认再买入</li>
        <li>更新：买入规则增加均线条件</li>
      </ul>
    </div>
  </div>
</div>

<h2 id="section-9-5-3">🎯 9.5.3 目标管理</h2>

### SMART目标设定

<div class="tip-box">
  <h4>🎯 SMART目标原则详解</h4>
  <ul>
    <li><strong>S - Specific（具体）：</strong>目标要明确具体，如"年化收益15%"而非"多赚钱"</li>
    <li><strong>M - Measurable（可衡量）：</strong>目标要可量化，能够客观评估</li>
    <li><strong>A - Achievable（可实现）：</strong>目标要合理，不要设定不切实际的预期</li>
    <li><strong>R - Relevant（相关）：</strong>目标要与投资策略和能力匹配</li>
    <li><strong>T - Time-bound（有时限）：</strong>目标要有明确的时间框架</li>
  </ul>
</div>

### 习惯养成方法

<div class="tip-box">
  <h4>📝 21天习惯养成法</h4>
  <ol>
    <li><strong>明确目标：</strong>选择一个要养成的习惯</li>
    <li><strong>设定触发：</strong>绑定已有习惯（如早餐后看新闻）</li>
    <li><strong>从小开始：</strong>先从最小行动开始</li>
    <li><strong>记录打卡：</strong>每天记录完成情况</li>
    <li><strong>奖励自己：</strong>连续完成给予奖励</li>
    <li><strong>坚持21天：</strong>21天后习惯初步形成</li>
    <li><strong>持续90天：</strong>90天后习惯稳固</li>
  </ol>
