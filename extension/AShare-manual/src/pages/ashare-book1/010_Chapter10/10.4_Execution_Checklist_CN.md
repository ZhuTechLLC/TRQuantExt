<h2 id="section-10-4-1">✅ 10.4.1 买入检查清单</h2>

### 买入前必须确认的事项

<div class="tip-box">
  <h4>✅ 买入检查清单（完整版）</h4>
  <p><strong>一、投资逻辑确认</strong></p>
  <ul>
    <li>□ 能否用一句话说清楚买入理由？</li>
    <li>□ 投资逻辑是否有数据支撑？</li>
    <li>□ 是否在自己的能力圈内？</li>
    <li>□ 是否受到市场情绪或他人影响？</li>
  </ul>
  <p><strong>二、公司质量确认</strong></p>
  <ul>
    <li>□ ROE是否>15%且稳定？</li>
    <li>□ 净利润增速是否>20%？</li>
    <li>□ 资产负债率是否<60%？</li>
    <li>□ 经营现金流是否健康？</li>
    <li>□ 管理层是否诚信可靠？</li>
    <li>□ 是否有护城河？</li>
  </ul>
  <p><strong>三、估值确认</strong></p>
  <ul>
    <li>□ PE分位是否<50%？</li>
    <li>□ PEG是否<1（成长股）？</li>
    <li>□ 是否有安全边际？</li>
  </ul>
  <p><strong>四、仓位确认</strong></p>
  <ul>
    <li>□ 买入后单只仓位是否<15%？</li>
    <li>□ 买入后行业仓位是否<30%？</li>
    <li>□ 是否分批建仓？</li>
  </ul>
  <p><strong>五、风险确认</strong></p>
  <ul>
    <li>□ 止损位是否已设定？</li>
    <li>□ 目标价是否已确定？</li>
    <li>□ 最坏情况能否承受？</li>
  </ul>
</div>

### 买入前"三问"

<div class="board-details">
  <div class="board-card">
    <h4>❓ 买入前必问三个问题</h4>
    <div class="board-content">
      <ol>
        <li><strong>为什么买？</strong>——投资逻辑是什么？</li>
        <li><strong>什么时候卖？</strong>——止盈止损条件是什么？</li>
        <li><strong>如果错了怎么办？</strong>——最坏情况的应对？</li>
      </ol>
      <p><strong>如果回答不清楚，不要买入！</strong></p>
    </div>
  </div>
</div>

### 季度检查清单

<div class="tip-box">
  <h4>📋 季度持仓检查清单</h4>
  <ul>
    <li>□ 每只持仓的投资逻辑是否依然成立？</li>
    <li>□ 季报业绩是否符合预期？</li>
    <li>□ 估值是否仍然合理？</li>

<h2 id="section-10-4-3">🔄 10.4.3 卖出检查清单</h2>

### 卖出前确认事项

<div class="tip-box">
  <h4>🔄 卖出检查清单（完整版）</h4>
  <p><strong>一、卖出理由确认</strong></p>
  <ul>
    <li>□ 是否触发止盈条件（达到目标价/估值过高）？</li>
    <li>□ 是否触发止损条件（亏损8%/跌破支撑）？</li>
    <li>□ 投资逻辑是否被破坏？</li>
    <li>□ 是否有更好的投资机会需要资金？</li>
  </ul>
  <p><strong>二、情绪确认</strong></p>
  <ul>
    <li>□ 是否因为恐慌而卖出？</li>
    <li>□ 是否因为短期波动而卖出？</li>
    <li>□ 是否受到他人影响而卖出？</li>
    <li>□ 卖出决策是否理性客观？</li>
  </ul>
  <p><strong>三、记录确认</strong></p>
  <ul>
    <li>□ 卖出理由是否已记录？</li>
    <li>□ 盈亏情况是否已计算？</li>
    <li>□ 经验教训是否已总结？</li>
  </ul>
  <p><strong>四、后续安排</strong></p>
  <ul>
    <li>□ 资金去向是否已规划？</li>
    <li>□ 是否需要调整其他持仓？</li>
  </ul>
</div>

      </tr>

## 💎 核心要点总结

<div class="key-points">
  <div class="key-point">
    <h4>✅ 买入有清单</h4>
    <p>每次买入前对照清单，确保符合系统要求，不冲动买入。</p>
  </div>
  <div class="key-point">
    <h4>📋 持有定期查</h4>
    <p>定期检查持仓，及时发现问题，不当甩手掌柜。</p>
  </div>
  <div class="key-point">
    <h4>🔄 卖出要确认</h4>
    <p>卖出前确认理由，避免情绪化操作，做到理性卖出。</p>
  </div>
  <div class="key-point">
    <h4>📝 记录要完整</h4>
    <p>完整记录每笔交易，为复盘和改进提供依据。</p>
  </div>
