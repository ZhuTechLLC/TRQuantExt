> **核心摘要：**

### PS因子的经济学逻辑

<div class="tip-box">
  <h4>💡 为什么使用PS？</h4>
  <ul>
    <li><strong>营收更稳定：</strong>营收波动小于利润，估值更稳定</li>
    <li><strong>不受会计处理影响：</strong>营收较难被操纵</li>
    <li><strong>适用亏损公司：</strong>亏损公司PE无意义，但PS仍可计算</li>
    <li><strong>成长股估值：</strong>高成长公司利润率低但营收增速快</li>
    <li><strong>参考：</strong>第一册5.1节估值基础</li>
  </ul>

### PS因子不适用场景

<div class="tip-box">
  <h4>⚠️ PS因子的局限性</h4>
  <ul>
    <li><strong>高利润率行业：</strong>如白酒、软件，PE更有意义</li>
    <li><strong>周期性行业：</strong>营收波动大，PS不稳定</li>
    <li><strong>金融行业：</strong>营收定义特殊，不适用PS</li>
    <li><strong>营收质量差：</strong>虚增营收的公司PS失真</li>
  </ul>
</div>

<h2 id="implementation">🖥️ 策略实现</h2>

### 韬睿平台PS因子计算

### PS因子计算代码

```python
def calc_ps_factor(stocks, date):
    """
    计算PS因子
    
    参数:
        stocks: 股票列表
        date: 计算日期
    
    返回:
        PS因子DataFrame
    """
    # 获取市值和营收数据
    df = get_fundamentals(query(
        valuation.code,
        valuation.market_cap,
        valuation.ps_ratio
    ).filter(
        valuation.code.in_(stocks),
        valuation.ps_ratio > 0  # 排除负PS
    ), date=date)
    
    # 计算PS倒数（低PS为好，取倒数后高值为好）
    df['ps_inv'] = 1 / df['ps_ratio']
    
    return df[['code', 'ps_ratio', 'ps_inv']]
```

### 低PS高成长策略

```python
def low_ps_growth_strategy(date, hold_num=30):
    """
    低PS高成长策略
    
    筛选条件：
    1. PS < 行业中位数
    2. 营收增速 > 30%
    3. 毛利率 > 20%
    """
    # 获取科技行业股票
    tech_stocks = get_industry_stocks(['I64', 'I65'], date)  # 软件、互联网
    
    # 获取PS和营收增速
    df = get_fundamentals(query(
        valuation.code,
        valuation.ps_ratio,
        indicator.inc_revenue_year_on_year,
        indicator.gross_profit_margin
    ).filter(
        valuation.code.in_(tech_stocks),
        valuation.ps_ratio > 0,
        indicator.inc_revenue_year_on_year > 30,
        indicator.gross_profit_margin > 20
    ), date=date)
    
    # 按PS排序（低PS优先）
    df = df.sort_values('ps_ratio')
    
    return df.head(hold_num)['code'].tolist()
```
