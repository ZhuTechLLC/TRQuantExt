<h2 id="definition">📊 股息率的定义与计算</h2>

### 股息率的基本定义

<div class="tip-box">
  <h4>📊 股息率（Dividend Yield）</h4>
  <ul>
    <li><strong>基本公式：</strong>股息率 = 每股股息 / 股价 = 分红总额 / 市值</li>
    <li><strong>经济含义：</strong>投资者每年从股息中获得的现金回报率</li>
    <li><strong>与PE的关系：</strong>股息率 = 分红率 / PE</li>
  </ul>
</div>

### 股息率因子计算代码

```python
def calculate_dividend_yield(stocks, date):
    """
    计算股息率因子
    
    参数:
        stocks: 股票列表
        date: 计算日期
    
    返回:
        股息率因子Series
    """
    # 获取市值
    valuation = get_valuation(stocks, date, fields=['market_cap'])
    
    # 获取过去12个月分红总额
    q = query(
        finance.STK_DIVIDEND.code,
        finance.STK_DIVIDEND.dividend_per_share
    ).filter(
        finance.STK_DIVIDEND.code.in_(stocks)
    )
    dividend = finance.run_query(q)
    
    # 计算股息率 = 分红 / 市值
    div_yield = dividend['dividend_per_share'] / valuation['market_cap']
    
    return div_yield
```

<h2 id="strategy">🛡️ 高股息策略原理</h2>

### 高股息策略的逻辑

<div class="exchange-grid">
  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">💰</span>
      <h3>现金回报确定</h3>
    </div>
    <div class="exchange-content">
      <p><strong>逻辑：</strong>股息是真金白银的回报，不受股价波动影响</p>
      <p><strong>优势：</strong>即使股价下跌，仍有稳定现金流入</p>
    </div>
  </div>

  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">🛡️</span>
      <h3>防御属性</h3>
    </div>
    <div class="exchange-content">
      <p><strong>逻辑：</strong>高股息股票估值通常较低，下跌空间有限</p>
      <p><strong>表现：</strong>熊市中高股息策略往往跑赢大盘</p>
    </div>
  </div>

  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">📊</span>
      <h3>盈利质量信号</h3>
    </div>
    <div class="exchange-content">
      <p><strong>逻辑：</strong>能持续分红的公司通常盈利质量较好</p>
      <p><strong>参考：</strong>第一册5.4节现金流分析</p>
    </div>
  </div>
</div>

<h2 id="policy">📋 分红政策分析</h2>

### A股分红政策特点

<div class="tip-box">
  <h4>🇨🇳 A股分红政策</h4>
  <ul>
    <li><strong>监管要求：</strong>证监会鼓励上市公司分红，部分再融资与分红挂钩</li>
    <li><strong>分红率水平：</strong>A股平均分红率约30%，低于成熟市场（40-50%）</li>
    <li><strong>分红意愿：</strong>国企分红意愿较强，民企分红率差异大</li>
  </ul>

### 股息陷阱识别

<div class="tip-box">
  <h4>⚠️ 股息陷阱：高股息率的假象</h4>
  <p><strong>股息陷阱特征：</strong></p>
  <ul>
    <li><strong>股价暴跌导致：</strong>股价大跌后股息率被动升高</li>
    <li><strong>一次性分红：</strong>特别股息导致当年股息率异常高</li>
    <li><strong>分红不可持续：</strong>盈利下滑但维持高分红</li>
    <li><strong>借债分红：</strong>现金流不足但仍高分红</li>
  </ul>
</div>

**过滤条件：**
- 分红率 < 80%
- 经营现金流/分红 > 1.2
- 连续3年以上分红
- 近1年股价跌幅 < 30%
