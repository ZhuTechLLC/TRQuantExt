<h2 id="logic">💡 经济学逻辑</h2>

### 成长因子有效的三种解释

<div class="exchange-grid">
  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">📈</span>
      <h3>盈利增长假说</h3>
    </div>
    <div class="exchange-content">
      <p><strong>核心观点：</strong>高成长公司未来盈利增加，股价应该上涨</p>
      <p><strong>逻辑：</strong>股价 = 盈利 × 估值，盈利增长推动股价</p>
      <p><strong>公式：</strong>P = E × PE，E↑ → P↑</p>
    </div>
  </div>

  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">🎯</span>
      <h3>估值重估假说</h3>
    </div>
    <div class="exchange-content">
      <p><strong>核心观点：</strong>高成长公司应该享受更高估值</p>
      <p><strong>逻辑：</strong>PEG估值法，增速越高，合理PE越高</p>
      <p><strong>公式：</strong>合理PE = 增速 × PEG</p>
    </div>
  </div>

  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">🧠</span>
      <h3>行为金融假说</h3>
    </div>
    <div class="exchange-content">
      <p><strong>核心观点：</strong>投资者对成长的反应不足</p>
      <p><strong>偏差类型：</strong>锚定效应、保守主义</p>
      <p><strong>参考：</strong>第一册2.2节认知偏差</p>
    </div>
  </div>
</div>

### PEG估值法

<div class="tip-box">
  <h4>📊 PEG = PE / 增速</h4>
  <ul>
    <li><strong>PEG < 1：</strong>估值偏低，增速被低估</li>
    <li><strong>PEG = 1：</strong>估值合理</li>
    <li><strong>PEG > 2：</strong>估值偏高，需要警惕</li>
    <li><strong>适用范围：</strong>成长期公司，增速>15%</li>
  </ul>
</div>

### 成长因子计算示例

```python
def calculate_growth_factors(stocks, date):
    """
    计算成长因子
    
    参数:
        stocks: 股票列表
        date: 计算日期
    
    返回:
        成长因子DataFrame
    """
    # 获取财务指标
    q = query(
        indicator.code,
        indicator.inc_revenue_year_on_year,    # 营收同比增速
        indicator.inc_net_profit_year_on_year, # 净利润同比增速
        indicator.roe                           # ROE
    ).filter(indicator.code.in_(stocks))
    
    df = get_fundamentals(q, date)
    
    # 处理异常值
    df['revenue_growth'] = df['inc_revenue_year_on_year'].clip(-100, 500)
    df['profit_growth'] = df['inc_net_profit_year_on_year'].clip(-100, 500)
    
    return df
```

<h2 id="trap">⚠️ 成长陷阱识别</h2>

### 什么是成长陷阱

<div class="tip-box">
  <h4>⚠️ 成长陷阱：高增速的假象</h4>
  <p><strong>定义：</strong>公司看起来增速很高，但实际上成长不可持续或质量不高。</p>
  <p><strong>常见类型：</strong></p>
  <ul>
    <li><strong>低基数效应：</strong>去年基数低导致今年增速高</li>
    <li><strong>一次性因素：</strong>非经常性损益推高利润</li>
    <li><strong>并购堆砌：</strong>外延并购而非内生增长</li>
    <li><strong>透支未来：</strong>激进确认收入，透支未来业绩</li>
  </ul>
</div>
