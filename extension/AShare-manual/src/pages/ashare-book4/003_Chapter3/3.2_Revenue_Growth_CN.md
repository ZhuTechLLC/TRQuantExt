<h2 id="definition">📊 因子定义</h2>

### 营收增速计算方法

<div class="comparison-table">
  <table>
    <thead>
      <tr>
        <th>指标</th>
        <th>计算公式</th>
        <th>优点</th>
        <th>缺点</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td><strong>同比增速</strong></td>
        <td>(本期营收 - 去年同期) / 去年同期</td>
        <td>消除季节性</td>
        <td>滞后性强</td>
      </tr>
      <tr>
        <td><strong>环比增速</strong></td>
        <td>(本季营收 - 上季) / 上季</td>
        <td>时效性强</td>
        <td>季节性影响大</td>
      </tr>
      <tr>
        <td><strong>TTM增速</strong></td>
        <td>(TTM营收 - 去年TTM) / 去年TTM</td>
        <td>平滑波动</td>
        <td>反应较慢</td>
      </tr>
      <tr>
        <td><strong>3年复合增速</strong></td>
        <td>(当前/3年前)^(1/3) - 1</td>
        <td>反映长期趋势</td>
        <td>更新频率低</td>
      </tr>
    </tbody>
  </table>
</div>

### 营收增速的经济学逻辑

<div class="tip-box">
  <h4>💡 营收增速的重要性</h4>
  <ul>
    <li><strong>规模扩张：</strong>营收增长反映市场份额扩大或行业增长</li>
    <li><strong>盈利基础：</strong>营收是利润的基础，营收增长是利润增长的前提</li>
    <li><strong>估值支撑：</strong>高营收增速支撑高估值（PS估值法）</li>
    <li><strong>竞争力体现：</strong>持续高增长体现公司竞争优势</li>

### 韬睿平台营收增速因子计算

```python
def calc_revenue_growth(stocks, date):
    """
    计算营收增速因子
    
    参数:
        stocks: 股票列表
        date: 计算日期
    
    返回:
        营收增速DataFrame
    """
    # 获取营收增速数据
    df = get_fundamentals(query(
        valuation.code,
        indicator.inc_revenue_year_on_year
    ).filter(valuation.code.in_(stocks)), date=date)
    
    # 处理异常值（增速超过500%或低于-100%视为异常）
    df['revenue_growth'] = df['inc_revenue_year_on_year'].clip(-100, 500)
    
    return df[['code', 'revenue_growth']]

def calc_revenue_cagr(stock, date, years=3):
    """
    计算营收复合增速
    
    参数:
        stock: 股票代码
        date: 计算日期
        years: 计算年数
    
    返回:
        复合增速
    """
    # 获取历史年度营收
    revenues = []
    for i in range(years + 1):
        year_date = date - pd.DateOffset(years=i)
        rev = get_fundamentals(query(
            income.operating_revenue
        ).filter(valuation.code == stock), date=year_date)
        if len(rev) > 0:
            revenues.append(rev.iloc[0, 0])
    
    if len(revenues) == years + 1 and revenues[-1] > 0:
        cagr = (revenues[0] / revenues[-1]) ** (1/years) - 1
        return cagr
    return None

def high_growth_strategy(date, hold_num=30):
    """
    高营收增速策略
    
    筛选条件：
    1. 营收增速 > 25%
    2. 毛利率 > 25%（排除低质量增长）
    3. 营收增速加速（本期 > 上期）
    """
    stocks = get_index_stocks('000905.XSHG', date)  # 中证500
    
    df = get_fundamentals(query(
        valuation.code,
        indicator.inc_revenue_year_on_year,
        indicator.gross_profit_margin
    ).filter(
        valuation.code.in_(stocks),
        indicator.inc_revenue_year_on_year > 25,
        indicator.gross_profit_margin > 25
    ), date=date)
    
    # 按营收增速排序
    df = df.sort_values('inc_revenue_year_on_year', ascending=False)
    
    return df.head(hold_num)['code'].tolist()
```
