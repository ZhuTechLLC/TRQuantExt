### 动量效应的发现

<div class="tip-box">
  <h4>📖 Jegadeesh-Titman动量研究（1993）</h4>
  <ul>
    <li><strong>核心发现：</strong>过去3-12个月表现好的股票，未来3-12个月仍会表现好</li>
    <li><strong>策略构建：</strong>买入过去6个月收益最高的10%股票，卖出最低的10%</li>
    <li><strong>持有期：</strong>持有6个月后调仓</li>
    <li><strong>美股表现：</strong>年化超额收益约12%</li>
  </ul>

### 动量效应的解释

<div class="exchange-grid">
  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">🧠</span>
      <h3>反应不足假说</h3>
    </div>
    <div class="exchange-content">
      <p><strong>核心观点：</strong>投资者对新信息反应不足</p>
      <p><strong>机制：</strong>保守主义、锚定效应</p>
      <p><strong>结果：</strong>价格趋势延续</p>
    </div>
  </div>

  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">👥</span>
      <h3>羊群效应假说</h3>
    </div>
    <div class="exchange-content">
      <p><strong>核心观点：</strong>资金追逐热门股票</p>
      <p><strong>机制：</strong>正反馈交易、业绩追逐</p>
      <p><strong>结果：</strong>形成价格趋势</p>
    </div>
  </div>

  <div class="exchange-card">
    <div class="exchange-header">
      <span class="exchange-icon">📊</span>
      <h3>信息扩散假说</h3>
    </div>
    <div class="exchange-content">

<h2 id="ashare">🇨🇳 A股特殊性</h2>

### A股动量效应特点

<div class="tip-box">
  <h4>🇨🇳 A股动量的独特规律</h4>
  <p><strong>核心特点：</strong>短期反转强，中期动量弱</p>
  <ul>
    <li><strong>散户占比高：</strong>追涨杀跌后快速反转（参考第一册1.3节）</li>
    <li><strong>涨跌停制度：</strong>限制极端波动，加速反转</li>
    <li><strong>T+1制度：</strong>无法日内回转，短期反转更明显</li>
    <li><strong>政策影响大：</strong>政策驱动行情，趋势不稳定</li>
  </ul>
</div>

### 动量因子计算示例

```python
def calculate_momentum_factors(stocks, date):
    """
    计算动量因子
    
    参数:
        stocks: 股票列表
        date: 计算日期
    
    返回:
        动量因子DataFrame
    """
    import pandas as pd
    
    df = pd.DataFrame(index=stocks)
    
    # 1周反转
    price_1w = get_price(stocks, end_date=date, count=5, fields='close')
    df['reversal_1w'] = -(price_1w.iloc[-1] / price_1w.iloc[0] - 1)
    
    # 1月反转
    price_1m = get_price(stocks, end_date=date, count=20, fields='close')
    df['reversal_1m'] = -(price_1m.iloc[-1] / price_1m.iloc[0] - 1)
    
    # 6月动量
    price_6m = get_price(stocks, end_date=date, count=120, fields='close')
    df['momentum_6m'] = price_6m.iloc[-1] / price_6m.iloc[0] - 1
    
    return df
```

## 💎 核心要点总结

<div class="key-points">
  <div class="key-point">
    <h4>🚀 动量是经典因子</h4>
    <p>动量效应在全球市场普遍存在，是重要的技术面因子</p>
  </div>
  <div class="key-point">
    <h4>🇨🇳 A股短期反转强</h4>
    <p>A股散户占比高，短期反转效应比中期动量更显著</p>
  </div>
  <div class="key-point">
    <h4>⏰ 时间窗口很重要</h4>
    <p>不同时间窗口效应不同，A股推荐1周反转或3-6月动量</p>
  </div>
  <div class="key-point">
    <h4>🔗 与基本面因子组合</h4>
    <p>动量因子与价值、质量因子组合使用效果更好</p>
  </div>
</div>
