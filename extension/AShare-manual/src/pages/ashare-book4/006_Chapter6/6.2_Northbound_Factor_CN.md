<h2 id="section-6-2-1">🌏 6.2.1 北向资金简介</h2>

### 沪深港通机制

<div class="tip-box">
  <h4>🌏 北向资金（沪深港通）</h4>
  <p><strong>定义：</strong>境外投资者通过香港交易所投资A股的资金</p>
  <p><strong>开通时间：</strong></p>
  <ul>
    <li><strong>沪港通：</strong>2014年11月开通</li>
    <li><strong>深港通：</strong>2016年12月开通</li>
  </ul>
  <p><strong>数据披露：</strong></p>
  <ul>
    <li><strong>每日净买入：</strong>当日公布</li>
    <li><strong>个股持股：</strong>每日披露（收盘后）</li>
  </ul>
</div>

### 北向资金因子类型

```python
def calculate_northbound_factor(stocks, date, lookback=20):
    """计算北向资金因子"""
    # 获取北向持股数据
    # 因子1：持股比例
    holding_ratio = get_northbound_holding(stocks, date)
    
    # 因子2：持股比例变化（近N日）
    past_date = date - pd.DateOffset(days=lookback)
    past_holding = get_northbound_holding(stocks, past_date)
    holding_change = holding_ratio - past_holding
    
    # 因子3：净买入金额（近N日）
    net_buy = get_northbound_netbuy(stocks, past_date, date)
    
    return holding_ratio, holding_change, net_buy
```
