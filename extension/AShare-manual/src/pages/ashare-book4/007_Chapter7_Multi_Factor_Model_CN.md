# 第七章：多因子模型构建

> **核心摘要：**
> 
> 多因子模型是量化选股的核心方法论，将价值、成长、质量、动量、资金流等因子整合为完整的选股模型。本章系统讲解多因子模型的理论框架、因子合成方法、组合构建和风险控制，帮助投资者构建专业级的量化选股系统。
