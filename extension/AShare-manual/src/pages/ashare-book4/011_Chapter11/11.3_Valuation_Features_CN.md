    <h4>📊 PEG计算与应用</h4>
    <div class="board-content">

```python
# PEG计算
PEG = PE / 净利润增速（%）

# 示例
# 股票A: PE = 30倍, 净利润增速 = 40%
# PEG = 30 / 40 = 0.75 (较低，有吸引力)

# 股票B: PE = 50倍, 净利润增速 = 30%
# PEG = 50 / 30 = 1.67 (较高，需谨慎)

# 韬睿量化平台PEG筛选
def peg_screening(date):
    financials = get_fundamentals(query(
        valuation.code,
        valuation.pe_ratio,
        indicator.inc_net_profit_year_on_year
    ), date=date)
    
    financials['peg'] = financials['pe_ratio'] / financials['inc_net_profit_year_on_year']
    
    # 筛选PEG < 1.5的股票
    return financials[financials['peg'] < 1.5]
```

<div class="tip-box">
  <h4>💡 估值策略建议</h4>
  <ul>
    <li><strong>成长消化估值：</strong>高成长股可接受较高PE，关键看PEG</li>
    <li><strong>行业对比：</strong>估值要与同行业对比，避免跨行业比较</li>
    <li><strong>动态视角：</strong>用未来1年预期利润计算Forward PE</li>
    <li><strong>估值弹性：</strong>小市值高成长股估值弹性更大</li>
