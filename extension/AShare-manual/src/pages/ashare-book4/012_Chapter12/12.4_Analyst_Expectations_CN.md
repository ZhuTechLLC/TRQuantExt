## 📈 预期修正因子

<div class="board-details">

    <div class="board-content">

```python
# 韬睿量化平台预期修正因子
def analyst_revision_factor(stock, date):
    """
    计算分析师预期修正因子
    """
    # 获取分析师预期数据
    estimates = get_analyst_estimates(stock, date)
    
    # 当前一致预期
    current_eps = estimates['eps_mean'].iloc[-1]
    
    # 1个月前一致预期
    prev_eps = estimates['eps_mean'].iloc[-20] if len(estimates) > 20 else current_eps
    
    # 预期修正幅度
    revision = (current_eps - prev_eps) / abs(prev_eps) if prev_eps != 0 else 0
    
    # 覆盖分析师数量
    coverage = estimates['analyst_count'].iloc[-1]
    
    # 评级变化
    rating_change = estimates['rating_change'].iloc[-20:].sum()
    
    return {
        'eps_revision': revision,
        'coverage': coverage,
        'rating_change': rating_change
    }
```

## 📊 分析师因子有效性

<div class="tip-box">
  <h4>📈 分析师因子研究结论</h4>
  <ul>
    <li><strong>预期修正：</strong>上调盈利预测的股票短期表现显著优于下调</li>
