# Investment Manual - 投资手册系统使用说明

## 📚 系统概述

这是一个基于Astro构建的投资手册系统，包含：
- 📖 高倍股特征分析
- 💼 基础金融投资知识
- 🎯 量化策略理论知识
- 📊 市场分析框架

## 🚀 快速开始

### 启动开发服务器

```bash
cd investment-manual
npm run dev
```

访问: http://localhost:4321

### 构建生产版本

```bash
npm run build
npm run preview  # 预览构建结果
```

### 部署到GitHub Pages

```bash
npm run deploy
```

## 📁 目录结构

```
investment-manual/
├── docs/                 # 参考文档（PDF和Markdown）
├── src/
│   ├── pages/           # 页面文件（228个页面）
│   │   ├── book1/      # 第一册：高倍股分析
│   │   ├── book2/      # 第二册：宏观经济
│   │   ├── book3/      # 第三册：个股研究
│   │   └── book4/      # 第四册：技术分析
│   ├── components/      # Astro组件
│   ├── layouts/         # 布局模板
│   └── styles/          # 样式文件
├── public/              # 静态资源
│   ├── documents/       # 文档资源
│   └── images/          # 图片资源
└── scripts/            # 工具脚本
```

## 🔧 可用命令

- `npm run dev` - 启动开发服务器
- `npm run build` - 构建生产版本
- `npm run preview` - 预览构建结果
- `npm run deploy` - 部署到GitHub Pages
- `npm run pdf:all` - 生成所有PDF

## 💡 使用建议

1. **查看文档**: 访问 http://localhost:4321 浏览手册
2. **搜索内容**: 使用浏览器搜索功能查找特定主题
3. **参考PDF**: `docs/` 目录下有相关PDF文档
4. **更新内容**: 修改 `src/pages/` 中的Markdown文件

## 🔗 集成到量化研究流程

这个投资手册系统是整个量化研究流程的知识基础：

```
Investment Manual (知识基础)
    ↓
```
